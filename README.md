## 🔍 详细目录说明

### 1. `Labs/` - 课程实验  
| 目录名          | 内容描述                     | 关联课程       |
|-----------------|------------------------------|----------------|
| `Lab1_LinkedList` | 链表实现（增删查改）         | 数据结构       |
| `Lab2_Stack`     | 栈的应用（表达式求值）       | C语言程序设计  |
| `Lab3_Sort`      | 排序算法（冒泡/快排）        | 算法基础       |

> **提示**：每个实验目录包含：  
> - `.c`源码文件（如`linked_list.c`）  
> - `README.md`实验报告（原理+调试心得）  

### 2. `LeetCode/` - 刷题记录  
- 按题目分类（如`链表`、`动态规划`）  
- 文件命名格式：`[题号]_[题目名称].c`（例：`206_反转链表.c`）  
- 代码规范：  
  ```c
  // 文件开头标注核心思路
  /* 
   * 解法：双指针迭代
   * 时间复杂度：O(n)  空间复杂度：O(1)
   */
  ```

### 3. `Projects/` - 实战项目  
| 项目名          | 技术亮点                     | 状态   |
|-----------------|------------------------------|--------|
| `Tetris_Game`   | 基于NCurses的终端俄罗斯方块  | ✅ 完成 |
| `File_Manager`  | 递归遍历目录+文件操作        | 🔧 开发中 |

### 4. `Notes/` - 学习笔记  
- 语法难点（如指针、内存管理）  
- 调试技巧（GDB使用指南）  
- 参考书精要（如《C Primer Plus》笔记）

## 🛠️ 如何运行代码
1. **克隆仓库**  
   ```bash
   git clone https://github.com/你的用户名/C-language-code.git
   ```
2. **编译示例**（以链表实验为例）  
   ```bash
   cd Labs/Lab1_LinkedList
   gcc linked_list.c -o list_demo && ./list_demo
   ```
